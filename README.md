# SQL-Work-Samples

A collection of SQL scripts and data models developed for professional use cases. This repository showcases advanced logic, data transformations, and optimization techniques in **Snowflake**.

## Key Topics
- **Advanced Logic:** Recursive CTEs and complex window functions.
- **Supply Chain Analytics:** Inventory projections, supply & demand planning, and logistics models.
- **Explainable Data (XAI):** Transparent query structures designed for business auditability.
- **Performance:** Optimized Snowflake-specific SQL patterns.

*Note: All scripts have been anonymized and generalized for public sharing.*
